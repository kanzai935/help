WebアプリケーションはJavaやPython、Ruby等の言語を用いて実装されており、以下のような事が実現できます

 - Eコマースサイト
 - ホームページ
 - ブログサービス
 - ソーシャルネットワークサービス

ここではAlibaba CloudのITインフラサービスに焦点を当て、以下の組み合わせによるWebアプリケーションの実装手段を紹介します。  
そして本記事を読む事で、Alibaba Cloud上でのWebアプリケーションの現場開発に少しでもお役立てになれば幸いです。
